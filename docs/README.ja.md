# UID Link (ブラウザ拡張機能) - 日本語

**UID Link** は、**UID.ONE** 主権アイデンティティ（Sovereign Identity）エコシステム向けに設計された、高性能なクロスブラウザ拡張機能です。ブラウザ層のセキュリティエージェントとして機能し、クライアント側で直接、ゼロトラストポリシー、コンテキスト認識型DLP（データ損失防止）、およびリアルタイムの暗号化セッションバインドを実行します。

---

## 🛡️ セキュリティエンジン機能

### 1. 暗号化セッションバインド
セッションハイジャックやトークンの窃盗を防止します。認証が成功すると、UID Linkは以下の方法でセッションをロックダウンします。
* ローカルハードウェアデバイスの指紋（Fingerprint）を生成。
* アクティブな `sessionToken` に対するHMACバインド署名を作成。
* バックエンドAPI `/v1/auth/session-binding/register/` にバインド情報を安全に登録。
* 特権リクエストを許可する前に、ローカルで署名を検証。

### 2. オリジン検証（フィッシング対策シールド）
訪問したドメインを能動的にチェックし、UID.oneブランドを標的としたなりすまし、タイポスクワッティング、フィッシングの試みを検出します。
* `uid-one-login.com`、`uidone-secure.com`、`uid.one.evil.com` などの類似ドメインバリエーションを正規表現パターンで照合。
* 偽装ページが検出された場合、警告を示す公式の赤色バナーを自動的に挿入し、ユーザーが資格情報を入力するのを阻止します。

### 3. ブラウザ層データ損失防止（DLP）
クライアント側の操作を監視し、個人情報（PII）、認証情報、または機密文書の意図しない漏洩を防ぎます。
* **プレゼンテーションモード（Presentation Mode）:** `Alt+Shift+P` で切り替えます。Webページ上のすべての機密プレーンテキストパターン（電話番号、メールアドレス、国民身分証番号、クレジットカード番号など）を動的にぼかし、非表示にします。画面共有やキャプチャからの漏洩を防ぎます。マウスをホバーすると一時的に表示されます。
* **ファイルアップロードインターセプター:** ファイル入力（input）やドラッグ＆ドロップイベントを監視します。クライアント側でファイルを分析し、不安全なファイルをブロックして警告オーバーレイを表示します。
* **クリップボードインターセプター:** コピー＆ペーストイベントをインターセプトします。確認モーダルによって不安全な貼り付けをブロックし、機密データをコピーした際にはブラウザの通知で警告します。
* **フォームインターセプター:** フォームデータがブラウザのコンテキストから送信される前に、コンプライアンススキャンを実行します。

### 4. Cookie Guard & アンチトラッキング
ユーザーがCookie同意バナーで「すべて受け入れる」をクリックした場合でも、プライバシーを保護します。
* ページコンテキストでの `document.cookie` への書き込みをインターセプトし、トラッキング/広告Cookie（例: `_ga`、`_gid`、`_fbp`、`_fbc`、`hj*`、`cluid`）の登録を阻止します。
* 機密個人情報（メールアドレス、クレジットカード番号、JWT、APIキーなど）を含むCookieを検出してブロックします。
* 5秒ごとに定期的なクリーンアップを実行し、HTTPレスポンスヘッダー経由で設定された追跡用Cookieを消去します。

### 5. 画像メタデータ自動削除 (Dynamic EXIF & Stripper)
アップロードされる画像から、プライバシーを漏洩する可能性のあるメタデータを自動的に削除します。
* ファイル入力またはドラッグ＆ドロップによる画像（JPEG/JPG）のアップロードをインターセプトします。
* 送信前に、クライアント側でGPS座標、カメラ機種名、作成日時ヘッダー（APP1セグメント）を消去します。

### 6. OTP自己消滅 / 入力キャッシュのクリア
共有環境や侵害された環境での機密入力や確認コードの漏洩を防ぎます。
* フォーム送信時に、OTP、2FAコード、パスワード、クレジットカード番号フィールドを検出します。
* 送信完了の300ms後に、自動的に入力値を消去し、ブラウザのオートコンプリート履歴をクリアします。
* クリップボードに高機密な個人情報が含まれる場合、即座にクリップボードをクリアします。

### 7. Global Privacy Control (GPC) & DNT の適用
すべてのWebインタラクションにおいて、ユーザーの主権的なプライバシー権を主張します。
* すべてのWebページのグローバルウィンドウコンテキストに、標準のプライバシー信号（`navigator.globalPrivacyControl = true` および `navigator.doNotTrack = '1'`）を注入します。

### 8. ビューポートクリーナー & 保護シールド
* **通知ブロッカー:** 迷惑なサードパーティのプッシュ通知をブロックするため、サービスワーカーの登録と通知許可をインターセプトします。
* **ビューポートクリーナー:** 安全なページ上の不審なフローティング要素、オーバーレイ、および潜在的なキーロガーを非表示にして入力フィールドを保護します。
* **タブフォーカスぼかし (Tab Focus Blurring):** タブが完全に非表示または切り替えられたとき（`document.hidden`）にのみ、自動的にページ全体をぼかし、盗み見から保護します。

## 🚀 インストール方法

拡張機能をインストールするには、以下の2つのシンプルな方法のいずれかを選択してください。

### オプション1：ビルド済みパッケージのインストール（推奨・最も簡単）
1. 本リポジトリのルートから **`uid-link-chrome.zip`**（Chrome, Edge, Brave用）または **`uid-link-firefox.zip`**（Firefox用）をダウンロードします。
2. ダウンロードしたzipファイルをコンピュータ上のフォルダに解凍します。
3. ブラウザを開き、`chrome://extensions/`（Firefoxの場合は `about:debugging`）にアクセスします。
4. 右上の **「デベロッパーモード」** をオンにします。
5. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、解凍したフォルダを選択します。

---

### オプション2：ソースコードからのクローンとビルド
1. 本リポジトリをコンピュータにクローンします。
   ```bash
   git clone https://github.com/oneuid/uid-extension.git
   cd uid-extension
   ```
2. 依存関係をインストールしてコンパイルします。
   ```bash
   npm install
   npm run build
   ```
3. `chrome://extensions/` を開き、**「デベロッパーモード」**をオンにして、**「パッケージ化されていない拡張機能を読み込む」**をクリックし、ビルドされた **`dist/chrome/`** フォルダを選択します。

## 📂 Native Messaging 設定

OS連携機能（セキュアなハードウェアチェックなど）を検証するには、対応するOSのシステムディレクトリに Native Messaging ホストのマニフェストファイルがインストールされていることを確認してください。

* **Linux:** `~/.config/google-chrome/NativeMessagingHosts/`
* **macOS:** `~/Library/Application Support/Google/Chrome/NativeMessagingHosts/`
* **Windows:** レジストリキー `HKEY_CURRENT_USER\Software\Google\Chrome\NativeMessagingHosts\`
